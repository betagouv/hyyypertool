# End to End tests

## Usage

```sh
# Because [Bun](https://github.com/oven-sh/bun/releases/tag/bun-v1.0.26) does not support cypress.
$ pnpm install
# Run all the test in headless mode
$ pnpm test
# Open cypress in electron
$ pnpm open
```
